# Event Propagation Assignment (React + Vite)

## Project Description
This project is a React application built using Vite that demonstrates the concept of **event propagation (bubbling)** in the DOM.

The application contains a nested component structure with an outer container and an inner button. Each element has its own click event handler. The project shows how events normally bubble from child to parent and how to prevent this behavior using `event.stopPropagation()`.

---

## Key Concepts Demonstrated
- Event bubbling in the DOM
- React event handling with `onClick`
- Using `event.stopPropagation()` to control event flow
- Nested components and UI interaction control
